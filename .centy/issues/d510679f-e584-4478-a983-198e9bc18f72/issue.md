# Create PRDetail component for TUI

Create a PR detail view with full PR information display.

## Acceptance Criteria

- [ ] Create tui/components/domain/PRDetail.tsx
- [ ] Display: title, description (scrollable with d/u), status, priority, branches, linked issues, reviewers, timestamps
- [ ] [e] to navigate to edit PR
- [ ] [Esc/Backspace] to go back to list
- [ ] Keyboard scrolling for long descriptions
- [ ] Show status with appropriate color
